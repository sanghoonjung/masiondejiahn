# JIANE

A polished Next.js dinner-course site for a five-guest summer table.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Chef mode

- Long-press the JIANE logo for 1.2 seconds, or
- Open `/?chef=true`, or
- Click the tiny footer label.

Chef mode includes shopping checklists saved to localStorage, timeline, full recipes, and the hidden late-night course.

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Framework preset: Next.js.
4. Deploy with default settings.
